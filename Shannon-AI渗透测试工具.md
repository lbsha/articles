## 技术架构

### 4阶段多智能体协作

```
1️⃣ 攻击面侦察
   - 整合 Nmap、Subfinder、WhatWeb
   - 浏览器自动化探索
   - 生成完整攻击面地图

2️⃣ 漏洞分析
   - 多 Agent 并行
   - 专注 SQL注入、XSS、IDOR、SSRF 等
   - 白盒+黑盒融合

3️⃣ 漏洞利用验证
   - 执行真实攻击
   - 确认后才标记"已确认"
   - 误报率极低

4️⃣ 专业报告生成
   - 附带 PoC 代码
   - 可一键复现
```

## 快速上手

### 安装

```bash
# 1. 克隆项目
git clone https://github.com/KeygraphHQ/shannon
cd shannon

# 2. 配置 API Key
cp .env.example .env
# 编辑 .env，填入 ANTHROPIC_API_KEY

# 3. 克隆目标应用源码到本地
git clone https://github.com/org/target-app ./repos/my-repo

# 4. 一键启动渗透测试
./shannon start URL=https://target-app.example.com REPO=my-repo
```

### 查看日志

```bash
./shannon logs
```
